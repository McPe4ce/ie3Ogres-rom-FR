# Character Name Glossary (JP → French)

Constancy is mandatory: once a name is decided here, use it **everywhere**. The
convention is the **official European (English-dub) Inazuma Eleven names**, to
match the shipped v06 patch (which already renders マモル/円堂 as "Mark", etc.).
For a character not confidently mappable to an official name, use **Hepburn
romaji** and record it here so it stays constant. Add rows as new names appear
(esp. while translating `unitbase.STR` and `evet.pkb`).

Legend for **Confidence**: ✅ confident · ⚠️ likely, verify against in-game FR ·
🔤 romaji fallback (no confident official name).
